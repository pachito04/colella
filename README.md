# 🏥 Plataforma de Kinesiología - Lic. Federico Colella

Plataforma integral para gestión de turnos, historia clínica y landing page de alta conversión.

## 🛠 Tech Stack

- **Frontend:** Next.js 16 (Turbopack), Tailwind CSS, Shadcn UI, Framer Motion.
- **Backend:** Server Actions, Prisma ORM.
- **Database:** PostgreSQL (Servidor Remoto).
- **Auth:** Auth.js (v5) + Google OAuth.
- **Pagos:** Mercado Pago API.
- **State:** Zustand.
- **Automation:** n8n (para Bots de WhatsApp/Telegram - *Externo*).
